# Atlas — Sprint 5.0.4C — Enterprise Email Designer

## Escopo
- Galeria de templates com miniaturas reais do HTML enviado.
- Prévia alinhada ao modelo selecionado.
- Novo template institucional de boas-vindas.
- Novo convite para cadastro no Portal do Cliente.
- Token individual previsto pela variável `{{TOKEN_CONVITE}}`.
- Layout construído com tabelas, CSS inline e regras móveis para Gmail/Outlook.

## Homologação
1. Execute `configurarAtlasDataFoundation()`.
2. Publique nova versão do Web App mantendo a URL.
3. Abra Central de Comunicação e confirme as miniaturas.
4. Teste Boas-vindas e Convite para o Portal no Gmail desktop e celular.
5. O link individual será ativado plenamente na Sprint 5.0.6 — Portal do Cliente Premium.
